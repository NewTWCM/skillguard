## 🎯 結論 (扣成本後仍穩健)

### ✅ 扣成本後仍可交易的策略

1. **BTC FNG < 10 做多 30 日**: **72.7% 勝率, +6.48%/筆** (n=33)
   - 最強信號, 每年僅觸發 4-6 次
   - 成本僅吃掉 3pp 勝率
2. **BTC FNG < 15 做多 30 日**: 54.7% 勝率, +1.54%/筆 (n=148)
   - 邊際有效, 高頻但低報酬
3. **Banini「停損賣出」→ 做多 5 日**: 68.8% 勝率, +3.09%/筆 (n=16)
   - 小樣本但事件驅動信號強
4. **Peter Kurz 看空 → 做多 3 月**: 68.4% 勝率, +3.90%/筆 (n=12 看空)
   - 低頻但顯著正

### ❌ 扣成本後更慘的策略

1. **任何市場「貪婪做空」**: SPY 4.9% / BTC 11.5% 勝率, 平均 -30%+
   - 成本讓本已是災難的策略更災難
2. **Banini 看多→做空**: 33.3% 勝率, -1.82%/筆
   - 證實「反向操作她的買入」會虧錢

### 🔬 成本影響洞察

- **低頻 + 大持有**: 成本影響極小 (SPY 0 pp 降幅)
- **高頻信號**: 成本可能吃掉整個 alpha (台股 0.42% round-trip 很關鍵)
- **加密永續 30 日**: 成本 1.10% 會轉換部分臨界策略為負 EV
- **期權**: 1.03% round-trip 讓除非大幅度 (>10%) 否則難有淨正報酬

## 🧮 自訂成本重算範例

若你使用自己的券商:
```bash
python3 apply_costs_to_backtests.py --interactive
# 會依序詢問加密 / 美股 / 台股的:
#   commission_bps, commission_discount, tax_bps, slippage_bps_per_leg
# 按 Enter 保留預設, 或輸入自己的值
```

或以 module 呼叫:
```python
from costs import CostProfile, apply_costs

# 我的券商 (範例)
p = CostProfile(
    market="my_broker",
    commission_bps=14.25,
    commission_discount=0.6,  # 6 折 (非 2.8 折)
    tax_bps=30,
    slippage_bps_per_leg=5,   # 較差流動性
    rounds="buy_sell",
    tax_side="sell_only",
)
net = apply_costs(gross_ret_pct=3.0, profile=p)  # = 3.0% - (14.25*0.6*2 + 30 + 10)/100
```
